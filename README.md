# 🚀 Productivity Dashboard - To-Do List & Pomodoro Timer

A modern productivity web app that combines a **To-Do List manager** with a **Pomodoro timer** to help users organize tasks, maintain focus, and improve productivity.

## ✨ Features

### ✅ To-Do List
- Add new tasks
- Set task priority (High, Medium, Low)
- Search tasks
- Filter tasks:
  - All tasks
  - Completed tasks
  - Pending tasks
- Mark tasks as completed
- Track total tasks

### ⏱️ Pomodoro Timer
- 25-minute focus sessions
- Short break timer
- Long break timer
- Start / Pause / Reset controls
- Custom timer settings:
  - Work duration
  - Short break duration
  - Long break duration

### 📊 Productivity Tracking
- Completed task counter
- Pomodoro session tracking
- Focus time calculation
- Current productivity streak

## 🛠️ Technologies Used

- CSS3
- JavaScript
- Local Storage
